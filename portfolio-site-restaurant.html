Api pour l'exctraction de données

J’ai conçu et développé une API innovante dédiée à l'extraction de données à partir de CVs, combinant les capacités d'un OCR avancé (PaddleOCR) et d'un modèle de langage (LLM) pour une compréhension contextuelle et précise des informations.

L’objectif de cette API est de transformer le processus de sélection des candidatures en un flux de travail entièrement automatisé. En analysant chaque CV, l'API extrait des informations essentielles telles que l'expérience professionnelle, les compétences techniques, les formations, et plus encore, en les structurant en un format JSON clair et homogène.

Conçue pour s'adapter à des formats de CV variés, l'API intègre des fonctionnalités avancées pour gérer les différentes mises en page et les particularités du langage naturel utilisé dans les CVs. Elle s'intègre facilement dans les systèmes de gestion de ressources humaines (SIRH) pour accélérer et optimiser le processus de recrutement.

Technologies utilisées : PaddleOCR, LangChain, Python, Flask, Docker
