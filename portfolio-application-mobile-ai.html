Solution Rag pour la génération d'exercices

J’ai développé une solution RAG (Retrieval-Augmented Generation) innovante pour une application de génération d'exercices éducatifs.

L’objectif principal de cette solution est d'automatiser et de personnaliser la création d'exercices pour différents niveaux d'apprenants. Grâce à la génération de contenu assistée par récupération, cette application permet de produire des exercices ciblés, adaptés aux besoins des utilisateurs et en lien direct avec les exigences pédagogiques.

Cette solution offre une interface intuitive pour les éducateurs, facilitant l'accès à une bibliothèque d'exercices générés dynamiquement, ajustés pour différents niveaux scolaires, sujets et thèmes. Les éducateurs peuvent ainsi sélectionner, personnaliser, et utiliser immédiatement les exercices dans leurs cours ou examens, gagnant un temps précieux tout en enrichissant leur contenu pédagogique.

Technologies utilisées : Python, OpenAI API, LangChain, Chroma
